MoFi

Este dispositivo busca realizar una estrategia metodológica a través del juego y con uso de una caja interactiva, el cual aparte de ser un terapia innovadora es divertida para el niño; las actividades del tablero interactivo favorecen el uso de la mano y dedos de manera precisa, estimula la coordinación óculo manual, agarre, disociación de muñeca, ejercicios de fuerza, pinza digital, el terapeuta controla fácilmente al peluche robot mediante una aplicación móvil y supervisa la actividad que el usuario ejecuta.

Tablero Interactivo:

Es un sistema embebido que está conformado por sensores, un microcontrolador (Arduino mini mega pro), sistema de sonido, luces; todo esto debidamente armado en el interior de un tablero de madera, el cual es controlado desde un dispositivo móvil que se comunica con el tablero vía bluetooth.

  • Sombrero de Rosca
  • Broches
  • Cierre
  • Velcro
  • Cordones

Aplicacion Movil:

Aplicacion movil para el uso del tablero Interactivo:

  • Inicio de actividades
  • Visualizacion de informacion de ususario
  • Tiempos de actividad
  • Conección con el Servidor
