MoFi

Este dispositivo busca realizar una estrategia metodológica a través del juego y con uso de un juguete (oso de peluche), el cual aparte de ser un terapia innovadora es divertida para el niño; las actividades del peluche robot favorecen el uso de la mano y dedos de manera precisa, estimula la coordinación óculo manual, agarre, disociación de muñeca, ejercicios de fuerza, pinza d igital, el terapeuta controla fácilmente al peluche robot mediante una aplicación móvil y supervisa la actividad que el usuario ejecuta.

Peluche Robotico:

Es un sistema embebido que está conformado por sensores, un microcontrolador (Arduino mini mega pro), sistema de sonido, luces; todo esto debidamente armado en el interior de un muñeco de peluche, el cual es controlado desde un dispositivo móvil que se comunica con el peluche vía bluetooth, con el cual se puede realizar las siguientes actividades:

  • Sombrero de Rosca
  • Broches
  • Cierre
  • Manos de velcro
  • Cordones

Aplicacion Movil:

Aplicacion movil para el control del peluche robotico, la cual tiene las siguientes funciones:

  • Inicio de actividades
  • Visualizacion de informacion de ususario
  • Tiempos de actividad
  • Conección con servidor
